# 서울시 기온 예측

서울시 기온 데이터를 활용해 간편히 예측하고, 다년간 동일 기간의 실제 기온과 비교 차트를 제공하는 Flask 기반 웹 애플리케이션입니다.

## 주요 기능

### 1. 날짜별 일주일 기온 예측

- 사용자가 선택한 날짜를 기준으로 이후 7일간의 평균기온을 선형 회귀 모델로 예측합니다.
- 예측 결과는 Chart.js 선형 차트로 시각화되어 직관적으로 확인할 수 있습니다.

### 2. 다년간 실제 데이터 비교

- 서울시 기온 CSV(2021~현재)를 로드해 동일한 월·일 구간의 실제 평균기온을 자동으로 추출합니다.
- 예측선과 함께 과거 연도별 실측선이 함께 표시되어 계절성 및 추세를 비교할 수 있습니다.

### 3. 데이터 정제 및 모델 학습 자동화

- CSV 로딩 시 결측치를 제거하고 날짜를 `Timestamp` → `ordinal` 값으로 변환해 회귀 모델 학습에 적합한 형태로 가공합니다.
- 앱 실행 시 자동으로 학습을 수행하므로 별도 명령 없이 최신 데이터 기반 예측이 가능합니다.

### 4. 직관적인 웹 인터페이스

- 기본 날짜를 오늘로 설정해 즉시 예측을 시도할 수 있으며, 결과 페이지에서 다시 예측하기 링크를 통해 빠르게 돌아갈 수 있습니다.
- 연도별 범례는 최신 연도부터 정렬되어 한눈에 비교가 용이합니다.
- 예측 결과는 버튼 한 번으로 엑셀 파일(.xlsx)로 내려받아 추가 분석에 활용할 수 있습니다.

## 기술 스택

- **Backend**: Flask (Python), scikit-learn, pandas
- **Frontend**: HTML, CSS, JavaScript
- **시각화 라이브러리**: Chart.js

## 설치 및 실행

### 필수 패키지 설치

```bash
pip install flask pandas scikit-learn openpyxl
```

### 실행

```bash
python app.py
```

브라우저에서 `http://localhost:5000` 접속

## 사용 방법

1. **홈**: 날짜 선택 폼에서 예측 기준일을 지정합니다 (기본값은 내일).
2. **예측 요청**: "예측" 버튼을 누르면 선택한 날짜를 포함한 일주일 예측 결과 페이지로 이동합니다.
3. **결과 확인**: 예측선과 2022년 이후 연도별 실측선이 함께 표시된 선형 차트로 추세를 비교합니다.
4. **다시 예측**: "다시 예측하기" 링크를 눌러 다른 날짜로 반복 예측을 진행합니다.
